# How to Contribute

This is a very early and raw implementation of a contributors guide, so bear with me.

1. Clone repository to local machine.
2. npm install -g js-yaml
3. edit terraform.yaml with changes (the JSON file must not be edited directly)
4. js-yaml syntaxes/terraform.yaml > syntaxes/terraform.json
